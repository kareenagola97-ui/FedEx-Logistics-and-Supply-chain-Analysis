FedEx Logistics Dashboard & EDA Project

Project Overview

This project analyzes the SCMS Delivery History dataset using Python and Power BI.

The project consists of two parts:

1. Exploratory Data Analysis (EDA) using Python
2. Interactive Dashboard Development using Power BI

The goal is to understand logistics performance, shipment trends, vendor contribution, freight costs, and product performance.

Key Insights

- Air shipment mode contributes the highest freight cost.
- Revenue increased significantly over the years.
- A small number of vendors contribute a major portion of total revenue.
- Several countries account for most logistics activity.
- Product demand is concentrated among top-performing products.
